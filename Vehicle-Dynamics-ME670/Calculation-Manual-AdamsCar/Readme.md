
# **Vehicle Dynamics Calculations - Manual & Adams Car**
In this, the Acceleration, Braking, Ride and Low-Speed Steering Dynamics have been calculated in excel sheet by entering the parameters of Mahindra XUV500 vehicle. The results have been verified with ADAMS Car model. For more details about this project, please refer this [presentation](https://github.com/deepakraina99/VehicleDynamics_Adams/blob/master/src/Group%231_DeepakRaina_Nibodh_Tushar%40GIAN_VehicleDynamics.pdf)
> This work has been done as part of Vehicle Dynamics course at IIT Jodhpur.
#### System Specs:
- Windows/Ubuntu OS (32/64 bits)
- MS Excel
- Adams Car
